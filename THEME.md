# Theme Design: Mrs Philippines Montreal

Based on the `unbluredticket.gif`, the website theme will focus on a **luxurious, elegant, and professional** aesthetic, combining deep blacks with rich gold metallic accents.

## 1. Color Palette

| Usage | Color Name | Hex Code | Description |
| :--- | :--- | :--- | :--- |
| **Background** | Deep Charcoal | `#0A0A0A` | Primary background for the sleek, modern look. |
| **Primary Accent** | Metallic Gold | `#D4AF37` | Main color for headers, buttons, and borders. |
| **Light Accent** | Champagne Gold | `#F9E27E` | Highlights, hover states, and lighter text. |
| **Dark Accent** | Antique Gold | `#996515` | Shadows, secondary borders, and depth. |
| **Main Text** | Off-White | `#F5F5F5` | Primary reading text for high contrast on dark bg. |
| **Sub-Text** | Muted Gold | `#B59A5A` | Secondary information and labels. |

## 2. Typography

*   **Primary Font (Display):** Elegant Serif (e.g., *Playfair Display* or *Cinzel*) for titles and headings to evoke grace and prestige.
*   **Secondary Font (Body):** Clean Sans-Serif (e.g., *Montserrat* or *Lato*) for readability and modern feel.

## 3. Visual Elements

*   **Borders:** Thin gold borders (`1px` or `2px`) with subtle gradients.
*   **Gradients:** Use linear gradients for "Gold" elements to simulate metallic reflection (e.g., `linear-gradient(45deg, #996515 0%, #D4AF37 50%, #F9E27E 100%)`).
*   **Shadows:** Soft glow effects around primary buttons (Gold outer glow).
*   **Ornaments:** Subtle gold flourishes in corners or as section dividers, inspired by the ticket design.

## 4. Components Style

*   **Buttons:** Gold background with dark text, or dark background with gold border and gold text on hover.
*   **Cards:** Dark charcoal background with a very subtle gold border.
*   **Navigation:** Sticky header with a glassmorphism effect (semi-transparent black with gold bottom border).

## 5. Mobile-First Approach

*   Ensuring large touch targets for buttons.
*   Responsive typography that scales elegantly.
*   Full-width layouts on small screens, transitioning to contained layouts with margin on larger screens.
